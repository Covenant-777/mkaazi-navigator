# ⚖️ Mkaazi Navigator

**Legal Access for Kenyan Tenants & Property Buyers**

Mkaazi Navigator is a mobile-first legal tech application designed to help ordinary Kenyans understand their legal rights regarding tenancy disputes and property purchases.

## 🌟 Features

### 🏠 Tenant Protection
- **5 Tenant Scenarios**: Illegal lockout, utility shutoff, harassment, deposit disputes, rent increases
- **Interactive Q&A**: Simple questions with clear legal guidance
- **Multi-language**: English, Kiswahili, and Sheng support
- **PDF Generation**: Download professional legal notices

### 🏢 Property Buyer Calculator
- **Stamp Duty Calculation**: 4% urban / 2% rural rates
- **Cost Breakdown**: Stamp duty + legal fees + total estimate
- **Save Calculations**: History of all your property estimates

### 💾 Smart Features
- **Auto-save**: All calculations saved locally
- **History View**: Access past cases anytime
- **PDF Reports**: Professional legal documents with case numbers

## 🚀 Tech Stack

- **Frontend**: Next.js 14, React 18
- **Styling**: Custom CSS with gradient designs
- **PDF Generation**: @react-pdf/renderer
- **State Management**: React Hooks + Local Storage
- **Language Support**: 3 languages (EN, SW, SHENG)

## 📱 Quick Start

### Prerequisites
- Node.js 18+ or later
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/mkaazi-navigator.git

# Navigate to project
cd mkaazi-navigator

# Install dependencies
npm install

# Run development server
npm run dev
